Using DataCube


What is DataCube?


  • A data cube is a multi-dimensional representation of data that allows for efficient analysis and exploration of large datasets. It extends the concept of a two-dimensional table to multiple dimensions, enabling analysis across various dimensions simultaneously.

  • Data cubes organize data into a structured format that facilitates aggregations, summarizations, and drill-downs.

  • The data in a data cube is pre-aggregated, which means that it contains pre-computed summary information at different levels of granularity.

  • We use Xarray a python library to work on the datacube

What categories of data were included in our DataCube?


  • Satellite Images

    • We have collected a huge time range of Satellite images from the Adilabad area in Telangana for that we have added a product into our datacube and appended every single safe file by converting them into YAML file which are filled with the proper metadata of that area.

Usage of DataCube?


  • Data Storage:

    • Our datacube serves as a central repository for satellite images and CSV data. Satellite images are high-resolution captures obtained by satellite sensors, providing detailed geospatial information. Additionally, we store CSV files, which offer tabular data representation, further complementing our satellite imagery analysis efforts.

  • Flask App Development:

    • To enable seamless interaction with the datacube and present our findings, we have developed a Flask application. Flask, a web application development framework, empowers us to create a user-friendly interface and robust functionalities for analyzing and visualizing geospatial data.

  • NDVI and NDWI Analysis:

    • Our Flask app takes advantage of the data stored in the datacube to generate normalized difference vegetation index (NDVI) and normalized difference water index (NDWI) maps. NDVI measures vegetation health and density by analyzing the differences in reflectance between near-infrared and visible red bands. NDWI, on the other hand, assesses water content by examining the disparities in reflectance between shortwave infrared and green or near-infrared bands.
